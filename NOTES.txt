To get nginx to interpret files correctly:

        location /civicrm/civi.common.scot/ {
                rewrite ^/civicrm/civi.common.scot/(.*)$ /$1;
        }

        location /civi.common.scot/ {
                rewrite ^/civi.common.scot/(.*)$ /$1;
        }

In /etc/nginx/sites-enabled/default.conf
